# Sanctum Logo Assets

Place your logo files in this folder with the following names:

## Required Files

| File | Description | Usage |
|------|-------------|-------|
| `logo.png` | Main logo (400x400 or larger) | Web landing, general use |
| `logo-icon.png` | Icon only, small (64x64) | Favicon, small displays |
| `logo-white.png` | White/light version (if available) | Dark backgrounds |

## Recommended Sizes

- **logo.png**: 400x400px or larger (will be scaled down)
- **logo-icon.png**: 64x64px (favicon size)
- **logo-white.png**: Same as logo.png but for dark backgrounds

## Mobile Assets

For React Native/Expo, also place copies in:
- `mobile/assets/logo.png`
- `mobile/assets/icon.png` (for app icon)
- `mobile/assets/splash.png` (for splash screen)

## Color Reference

Based on your logo:
- Logo color: `#3D3D3D` (dark charcoal)
- Background (cream): `#F5F0E8`
- Background (white): `#FFFFFF`
